# ⬇ Install

### 1. Homebrew

{% code fullWidth="true" %}
```bash
# brew 다운로드
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 경로 추가
echo 'export PATH=/opt/homebrew/bin:$PATH' >> ~/.zshrc
source ~/.zshrc

# 설치 확인
brew --version
```
{% endcode %}
